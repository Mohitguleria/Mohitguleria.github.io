Computed property is a property declared as a method that reacts when the data(maybe one from the data returned object) it uses, is changed.
OR WE CAN SAY
Computed properties update, whenever the data they use, is updated, and doesn't simply become the part of re-rendering.
Computed properties are synchronous. We can not reach servers using computed properties and/or make network requests inside computed properties or wait to get a promise resolved. Similar to setTimeout.
It's advised not to make async requests or mutate the DOM inside a computed property getter!

whenever the data property changes, vue renders the template on the page again
